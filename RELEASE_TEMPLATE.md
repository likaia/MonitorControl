# Release Template

## Title

`LumaGlass 1.1.0`

## Suggested release text

```md
This release is built for macOS 26.

If you are not using macOS 26, please use the original MonitorControl project instead:
https://github.com/MonitorControl/MonitorControl

## Highlights

- Updated menu UI for macOS 26
- Native-style custom brightness HUD for macOS 26
- Improved slider behavior and menu layout on macOS 26
- Refined app branding and documentation

## Notes

- This fork is focused on macOS 26 compatibility and UX polish
- Behavior on earlier macOS versions is not the primary target of this fork

## Installation

1. Download the `.dmg` from this release
2. Move `LumaGlass.app` to `/Applications`
3. Launch the app and grant the requested permissions if prompted
```
