<h3>LumaGlass</h3>
<p>LumaGlass 是一个专门适配 macOS 26 的显示器控制工具，专注于外接显示器亮度与音量控制。
包含原生风格的菜单栏滑杆、键盘支持，以及针对 macOS 26 更新过的界面行为。</p>
